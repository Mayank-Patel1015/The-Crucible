# The Crucible

**Daily AI skill forge for clearer thinking.**

As AI systems become ubiquitous collaborators, the ability to think clearly alongside them has become a core professional and personal skill. The Crucible is the daily practice ground where structured missions train users to surface blind spots, steelman opposing views, and build lasting cognitive resilience rather than outsourcing judgment.

## About

Most tools focus on getting better outputs from models. The Crucible focuses on getting better thinking from the human.

Each day delivers one focused mission designed to train specific cognitive habits through deliberate prompting practice. Missions target real vulnerabilities in reasoning — starting with confirmation bias — by forcing you to surface blind spots, steelman opposing views, and practice intellectual honesty under structured pressure.

The current mission, **The Contrarian**, requires you to write a prompt that makes the AI genuinely argue against a position you hold, delivering three substantive counterarguments you had not already considered. The goal is not clever prompting tricks. It is building the habit of treating AI as a rigorous thinking partner rather than an echo chamber.

### Why this matters
AI amplifies both insight and illusion. Without deliberate practice, it is easy to outsource judgment and reinforce existing biases. The Crucible turns daily interaction with AI into repeatable training for clearer reasoning and stronger cognitive resilience.

### Current state
- Local-first web app with a focused, minimal interface
- Daily mission system with streak tracking
- Prompt input designed for serious practice (not casual chatting)
- Early missions emphasize contrarian thinking and bias resistance

The project is in active development. The first few missions are live and the core loop is working.

## Getting Started

Clone the repository and run locally:

```bash
git clone https://github.com/yourusername/the-crucible.git
cd the-crucible
# Follow setup instructions in the repo (or run the dev server)
npm install
npm run dev
```

Open http://localhost:3001/dashboard to begin.

## How It Works

1. A new mission becomes available each day.
2. Read the objective and craft a precise prompt.
3. Submit and review the AI’s response against the mission criteria.
4. Build your streak and track progress over time.

Missions are designed to create productive friction — not entertainment.

## Philosophy

The Crucible exists because prompting skill and thinking skill are not the same thing.  
We optimize for the latter.

- Treat AI as a sparring partner, not an oracle.
- Practice intellectual honesty daily.
- Build habits that transfer beyond any single model.
